# Xếp Lịch Làm Việc - Tiếng Anh

Ứng dụng web giúp xếp lịch nhân sự cho 2 cơ sở dạy tiếng Anh.

## Quy trình

1. **Cấu hình ca** — Thiết lập lịch làm, số trợ giảng cần mỗi ca
2. **Đăng ký lịch** — Một bảng duy nhất: điền tên TG đăng ký vào từng ô (vd: `Khang, Ý, H.Long`)
3. **Xếp lịch** — Tự động phân công dựa trên đăng ký

## Chạy ứng dụng

```bash
npm install
npm run dev
```

Mở http://localhost:5173
